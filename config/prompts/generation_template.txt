Adapta la estructura según el tipo de herramienta:

▸ SI ES IA GENERATIVA (Midjourney, DALL-E, ChatGPT, Suno, etc):
  1. Rol/Contexto del sistema

▸ SI ES SOFTWARE DE OFIMÁTICA (Word, Excel, PowerPoint, Google Docs, etc):

▸ SI ES CÓDIGO (Python, C++, VS Code, GitHub Copilot):
  1. Descripción funcional del programa
  2. Inputs y outputs esperados
  3. Lógica principal y algoritmo
  4. Librerías/frameworks sugeridos
  5. Estructura de archivos/módulos
  6. Manejo de errores y edge cases
  7. Ejemplos de uso

▸ SI ES DISEÑO/INFOGRAFÍA (Canva, Figma, Illustrator, InDesign):

  4. Paleta de colores DE MARCA (obligatoria si hay manual)
  5. Tipografía DE MARCA (obligatoria si hay manual)

  4. Relaciones y conexiones entre nodos

▸ SI ES EDICIÓN AUDIOVISUAL (Premiere Pro, After Effects, DaVinci Resolve, Logic Pro):

▸ SI ES 3D/ANIMACIÓN (Blender, Cinema 4D, Unity, Unreal Engine, AutoCAD):
  1. Descripción de la escena/modelo
  2. Geometría y topología
  3. Materiales y texturas (colores de marca si aplica)
  4. Iluminación y cámaras
  5. Animación (si aplica)

  7. Formato de exportación
